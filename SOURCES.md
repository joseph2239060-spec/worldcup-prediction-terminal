## ⚠️ 推断/估算的部分（中可信，需在视频里诚实标注）

这些不是官方数据，是基于上面真实数据的统计推断：

1. **预期进球 λ**：用近期战绩 + FIFA 排名差 + 主客场系数估算（Workflow 计算中）
2. **胜平负概率**：泊松分布从 λ 推算
3. **天气**：6/11 距核实日 3 天，预报可能仍变化
4. **赔率**：当前未接入实时盘口 API（The Odds API / Polymarket）。视频里若展示赔率必须实时抓取，不能用估算值

## 主要信息源清单（按可信度）

1. **FIFA 官网** fifa.com — 赛程、场馆、排名（最高）
2. **ESPN** espn.com — 赛前情报、首发预测、伤病
3. **Sky Sports / Goal / Olympics.com** — 球员名单、状态
4. **Sofascore** — FIFA 排名
5. **Wikipedia "2026 FIFA World Cup"** — 交叉验证
6. **Football-Data.co.uk** — 历史比分（算 λ 用）
7. **The Odds API / Polymarket** — 实时赔率（尚未接入）
